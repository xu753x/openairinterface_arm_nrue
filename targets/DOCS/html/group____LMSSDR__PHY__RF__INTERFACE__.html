int trx_lms_read ( openair0_device *  device,
openair0_timestamp *  ptimestamp,
void **  buff,
int  nsamps,
int  antenna_id 
)

Receive samples from hardware. Read nsamps samples from each channel to buffers. buff[0] is the array for the first channel. *ptimestamp is the time at which the first sample was received.

Parameters
devicethe hardware to use
[out]ptimestampthe time at which the first sample was received.
[out]buffAn array of pointers to buffers for received samples. The buffers must be large enough to hold the number of samples nsamps.
nsampsNumber of samples. One sample is 2 byte I + 2 byte Q => 4 byte.
antenna_idIndex of antenna port
Returns
number of samples read
